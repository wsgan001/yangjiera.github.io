  • Question: Launching the VM rebooted my pc (!).

    Answer: When you use Windows 10 Anniversary:
    1) Install Linux Subsystem (https://msdn.microsoft.com/en-us/commandline/wsl/install_guide)
    2) Install Ruby (sudo apt-get ruby)
    3) Install Twurl (sudo gem install twurl)
    4) Use twurl, authorize etc.
    (contributed by Thom Hubers)

  • While generating the tweets from “dev.twitter.com/streaming/reference/get/statuses/sample” I have got two types of data: “created_at” and “delete” (which contains very less information). Is this correct? And if yes, do we have to filter the “delete” data or count this as a tweet? (for Question 3, part 1.2)

    Answer: It means that the tweet was deleted. See reference here: status deletion notices delete
    The perfect way to handle this would be: you delete the previous tweet that the delete message refers to; and also delete the ‘delete message’ itself. But handling this is not the purpose of the exercise. You can pretend that nothing weird happens: just count every message you get as a tweet. Doing this way will not degrade your grade.

  • I used the coordinated of Amsterdam, how can I distinghuish the tweets from schiphol with slightly different coordinated, in python?

    Answer: To get the tweets sent from Schiphol, you have two options:
    1. use ‘json’ package to extract the coordinates of each tweets you get from Amsterdam— in real time, or after the completion of data retrieval — check if it is within the bounding box of Schiphol.
    2. change the coordinates parameter of your program to Schiphol coordinates, and run it again.
    I guess option 2 is easier. option 1 would be encouraged, but not required.
